# 7.5 获得ssl证书

## 准备

要申请一个ssl证书，需要你有一个完全控制权的域名，可以添加`A`、`AAAA`、txt记录。

### 获得域名
目前[freenom](https://www.freenom.com/)提供了了免费的域名，通过dnspod获得解析。
